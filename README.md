# KashtaKala App

KashtaKala is an Android application built with **Kotlin** and **Jetpack Compose** to help artisans and micro‑businesses manage their work.  
It provides tools for **project estimation, saving quotes, portfolio management, and catalog browsing**.

---

## ✨ Features
- 🔐 **Google Login** for secure access
- 📐 **Estimator**: Calculate area & volume, save quotes
- 📝 **Quotes**: View, delete, and manage saved quotes
- 🎨 **Portfolio**: Add projects with details, scrollable list
- 📂 **Catalog**: Placeholder for product listings
- 🚪 **Logout** option in bottom navigation

---

## 🛠 Tech Stack
- Android Studio
- Kotlin
- Jetpack Compose
- Room Database (for quotes)
- Google Sign‑In API
